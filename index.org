#+begin_h-content
- In Emacs org-mode, a list item is defined with a dash symbol (=-=) at the beginning.
- Subsequent items are defined in the same way.
- Other symbols include: =+= and numbers (/e.g./ =1.=, =1)=).
  - Numbers will result in an ordered list (=<ol>=) when exported to HTML
- A list item can be moved up and down with the keyboard combination {{{icon(keyboard)}}} =Meta= + =up= or =down=
  - The =Meta= key refers to the =Option= or =Alt= keys on a Mac or Windows keyboards, respectively.
